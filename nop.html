NOP - No Operation (90h)



	Usage:	NOP
	Modifies flags: None


	This is a do nothing instruction.  It results in occupation of both
	space and time and is most useful for patching code segments.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	none		  3	3     3	    1		  1
